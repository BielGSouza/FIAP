# Repositório da Faculdade - Engenharia de Software FIAP

Bem-vindo(a) ao meu repositório pessoal de projetos desenvolvidos durante o curso de Engenharia de Software. Este espaço é dedicado a consolidar e documentar todo o aprendizado prático e teórico adquirido ao longo da minha jornada acadêmica.

📚 Sobre o Curso

O curso de Engenharia de Software foca no desenvolvimento de sistemas de software de alta qualidade, abrangendo desde a concepção e design até a implementação, teste e manutenção. Este repositório reflete a aplicação desses princípios em diversos projetos práticos.

📂 Estrutura do Repositório

Este repositório está organizado para facilitar a navegação e o acesso aos projetos. Cada pasta principal representa um semestre ou uma área de estudo, e dentro dela, você encontrará subpastas para cada projeto específico.
